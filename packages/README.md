# HadesOS Packages

HadesOS packages are shared product boundaries for future runtime, connector, safety-kernel, UI, and cloud work. This repository currently keeps packages documentation-only until the implementation stack is selected.

## Packages

- [`contracts`](contracts/README.md): versioned, machine-readable JSON Schema contracts and typed examples for safety-kernel actions, approvals, audit events, rollback metadata, and connector manifests.

No package manager, runtime, generated files, or framework is defined in this layer yet.
