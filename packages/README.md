# HadesOS Packages

HadesOS packages are shared product boundaries for future runtime, connector, safety-kernel, UI, model-router, memory, and cloud work. This repository currently keeps packages documentation-only until the implementation stack is selected.

## Packages

- [`contracts`](contracts/README.md): versioned, machine-readable JSON Schema contracts and synthetic examples for safety-kernel action proposals, approval requests, audit events, rollback metadata, and connector manifests.

## Current package posture

No package manager, runtime, generated files, dependencies, framework, validation script, or CI workflow is defined in this layer yet. The contracts package is schema-only and exists to keep documentation and future implementations aligned.
