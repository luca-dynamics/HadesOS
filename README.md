# HadesOS

**HadesOS** is an ambient AI system app by Luca Dynamics for personal, home, office, and enterprise environments.

HadesOS helps users understand, automate, secure, and control their devices, spaces, systems, and workflows through a calm Apple-level desktop and mobile experience.

## Product Direction

HadesOS is not a replacement operating system. It is a system-level AI application layer that runs across desktop, mobile, and enterprise environments.

It connects to:

- Personal devices
- Files and apps
- Smart home systems
- Office devices
- Enterprise tools
- Sensors and cameras
- Security systems
- Automation workflows

## Core Principles

- Calm and simple user experience
- Privacy-first architecture
- Permission-based automation
- Human approval for sensitive actions
- Local-first where possible
- Enterprise-ready security
- Transparent audit history
- Safe AI execution

## Product Layers

### Personal Hades

For normal users managing personal devices, homes, files, apps, and automations.

### Hades Business

For teams and small companies managing office devices, shared spaces, policies, and incidents.

### Hades Enterprise

For organizations needing role-based access, compliance logs, device intelligence, security workflows, and enterprise integrations.

## Architecture

HadesOS is built around:

- Hades Runtime
- Context Engine
- Agent Orchestrator
- Model Router
- Safety & Permission Kernel
- Device and Space Connectors
- Automation Executor
- Audit Layer
